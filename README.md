# Vamos rachar

## Intuito

Projeto para a disciplina de Desenvolvimento de Software para Dispositivos Móveis.

## Equipe

- [@saulo-araujo](https://github.com/teste)

## Requisitos

- [x] O aplicativo deve ter dois inputs, um para a quantidade de pessoas, e outro para o valor a ser "rachado"(dividido)
- [x] Ele deve ter icones
- [x] Deve permitir o compartilhamento do valor final, para wpp/insta/etc
- [x] Deve falar o valor calculado usando TTS
- [x] O usuário não precisa clicar para calcular, ele já faz automaticamente após o preenchimento dos campos de valor e número de pessoa
